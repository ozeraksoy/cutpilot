# Changelog

All notable changes to Cutpilot will be documented in this file.

## [1.0.0] - 2026-05-04

### Added
- Whisper transcription with word-level timestamps (15+ languages)
- Frame-perfect Jump Cut with re-import strategy and filler word detection
- Auto-resync subtitles to Jump Cut sequence
- Multi-clip and batch transcription support
- AI Viral Shorts analysis with GPT-4o-mini / GPT-4o
- Automatic 9:16 sequence generation
- Inline viral candidate UI with checkbox selection
- Settings panel: silence threshold, custom filler words, AI model, candidate count, duration range
- Bilingual UI (Turkish / English)
- Persistent settings via localStorage
- macOS and Windows install scripts
